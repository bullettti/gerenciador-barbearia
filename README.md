# Gerenciador de Agendamentos de Barbearia

Sistema web para gerenciar agendamentos de uma barbearia, desenvolvido durante o desafio "14 Dias do Zero ao Deploy".

## Tecnologias utilizadas

### Backend

- Node.js
- Express
- PostgreSQL
- Jest
- Supertest

### Frontend

- React
- Vite

## Funcionalidades

- Criar agendamentos
- Listar agendamentos
- Cancelar agendamentos
- Listar horários disponíveis
- Bloquear horários já ocupados
- Validação de campos obrigatórios
- Logs básicos com níveis INFO, WARN e ERROR
- Testes automatizados no backend
